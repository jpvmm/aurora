Restructure Aurora's CLI command surface from 7+ top-level commands down to 4 core entry points (ask, chat, status, doctor) plus a config namespace.    - Also add the sub-typers: `deprecated_kb_app.add_typer(kb_config_app, name="config")` and `deprecated_kb_app.add_typer(kb_scheduler_app, name="scheduler")`

2. `deprecated_model_app`: same pattern with `typer.Typer(no_args_is_help=True, help="[DEPRECADO] Use \`aurora config model\`.")`:
   - Callback emits: `"Aviso: \`aurora model\` foi movido. Use \`aurora config model ...\`."`
   - Re-register: `model_set_command`, `model_start_command`, `model_stop_command`, `model_status_command`, `model_health_command`

3. `deprecated_memory_app`: same pattern with `typer.Typer(no_args_is_help=True, help="[DEPRECADO] Use \`aurora config memory\`.")`:
   - Callback emits: `"Aviso: \`aurora memory\` foi movido. Use \`aurora config memory ...\`."`
   - Re-register: `memory_list`, `memory_search`, `memory_edit`, `memory_clear`

Import pattern for deprecated.py:
