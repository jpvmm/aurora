The main command:
```python
@status_app.callback()
def status_command(
    ctx: typer.Context,
    json_output: bool = typer.Option(False, "--json", help="Renderiza resposta estruturada em JSON."),

        return
    _run_status(json_output=json_output)
```

The `_run_status` function collects state from 4 domains, each wrapped in try/except for graceful degradation:

1. **Version** — `importlib.metadata.version("aurora")`. Fallback: `"desconhecido"`.

2. **Model state** (per D-06: report-only, no side effects):
   - Import `ServerLifecycleService` lazily inside function
   - Call `ServerLifecycleService().get_status()` (reads lock file only — NO `check_health()`)
   - Extract: `lifecycle_state`, `model_id`, `endpoint_url`, `pid`, `uptime_seconds`
   - On exception: set all to None/fallback values, do NOT re-raise

3. **KB state**:
   - Import `load_kb_manifest` lazily
   - Import `load_settings` for vault_path and collection name
   - Call `load_kb_manifest()` — returns `KBManifest | None`
   - If manifest exists: `note_count = len(manifest.notes)`, `vault_root = manifest.vault_root`
   - For `last_update`: use manifest file mtime via `os.path.getmtime(get_kb_manifest_path())` formatted as ISO
   - If None: `note_count = 0`, `vault_root = settings.kb_vault_path or "(nao configurado)"`
   - On exception: graceful fallback

4. **Memory state**:
   - Import `EpisodicMemoryStore` lazily
   - Call `EpisodicMemoryStore().list_memories()`
   - `memory_count = len(memories)`
   - `last_session = memories[-1]["date"] if memories else None`
   - On exception: graceful fallback

5. **Config summary**: from `load_settings()`:
   - `vault_path`, `local_only`, `telemetry_enabled`, `kb_qmd_collection_name`

**JSON output** (per D-07): when `json_output=True`, emit a single JSON object:

**Add new check functions** — each returns `DoctorIssue | None`:

1. `_check_qmd_binary() -> DoctorIssue | None`:
   - `shutil.which("qmd")` — if None, return `DoctorIssue(category="qmd_missing", message="QMD nao encontrado no PATH.", commands=("pip install qmd",))`

2. `_check_qmd_version() -> DoctorIssue | None`:
   - Skip if `shutil.which("qmd")` is None (already caught by qmd_missing)
   - `subprocess.run(["qmd", "--version"], capture_output=True, text=True, timeout=10)`
   - If returncode != 0: return `DoctorIssue(category="qmd_version", message="QMD instalado mas nao responde a --version.", commands=("pip install --upgrade qmd",))`
   - Otherwise: no issue (version info can be logged to output later)

3. `_check_kb_collection(settings: RuntimeSettings) -> DoctorIssue | None`:
   - Import `load_kb_manifest` from `aurora.kb.manifest`
   - Try `manifest = load_kb_manifest()`
   - If `manifest is None`: return `DoctorIssue(category="kb_no_manifest", message="Manifesto KB nao encontrado. Execute a indexacao inicial.", commands=("aurora config kb ingest <caminho-do-vault>",))`
   - If `len(manifest.notes) == 0`: return `DoctorIssue(category="kb_collection_empty", message="KB sem notas indexadas. Execute a indexacao.", commands=("aurora config kb ingest <caminho-do-vault>",))`
   - On any exception: return `DoctorIssue(category="kb_manifest_error", message=f"Erro ao ler manifesto KB: {error}", commands=("aurora config kb rebuild",))`

4. `_check_kb_embeddings(settings: RuntimeSettings) -> DoctorIssue | None`:
   - Skip if `shutil.which("qmd")` is None
   - Run `subprocess.run(["qmd", "--index", settings.kb_qmd_index_name, "collection", "list"], capture_output=True, text=True, timeout=15)`
   - If `settings.kb_qmd_collection_name` NOT in stdout: return `DoctorIssue(category="kb_embeddings_missing", message="KB sem embeddings. Collection QMD nao encontrada.", commands=("aurora config kb rebuild",))`
   - On exception: return issue with recovery

5. `_check_memory_index(settings: RuntimeSettings) -> DoctorIssue | None`:
   - Import `EpisodicMemoryStore` from `aurora.memory.store`
   - Try `memories = EpisodicMemoryStore().list_memories()`
   - If no memories: return None (no memories is valid — user just hasn't chatted yet)
   - If memories exist, check QMD collection: `subprocess.run(["qmd", "--index", settings.kb_qmd_index_name, "collection", "list"], capture_output=True, text=True, timeout=15)`
   - If `"aurora-memory"` NOT in stdout and len(memories) > 0: return `DoctorIssue(category="memory_index_missing", message=f"Memorias encontradas ({len(memories)}) mas colecao QMD aurora-memory ausente.", commands=("aurora config memory clear --yes",))`
   - On exception: graceful, return None (memory is optional)

6. `_check_disk_space() -> DoctorIssue | None`:
   - Import `get_settings_path` from `aurora.runtime.paths`
   - `config_dir = get_settings_path().parent`
   - `usage = shutil.disk_usage(config_dir)`
   - If `usage.free < 500 * 1024 * 1024` (500 MB): return `DoctorIssue(category="disk_space_low", message=f"Espaco em disco baixo: {usage.free // (1024*1024)} MB livre em {config_dir}.", commands=(f"Libere espaco em {config_dir}",))`

7. `_check_python_version() -> DoctorIssue | None`:
   - If `sys.version_info < (3, 13)`: return `DoctorIssue(category="python_version", message=f"Python {sys.version_info.major}.{sys.version_info.minor} detectado. Aurora requer >= 3.13.", commands=("Instale Python 3.13+",))`

8. `_check_required_packages() -> list[DoctorIssue]`:
   - Check these packages: `"typer"`, `"pydantic"`, `"pydantic-settings"`, `"pyyaml"`, `"httpx"`
   - For each, try `importlib.metadata.version(name)` — if `PackageNotFoundError`, append `DoctorIssue(category="package_missing", message=f"Pacote '{name}' nao instalado.", commands=(f"pip install {name}",))`
   - Return list (may be empty)
