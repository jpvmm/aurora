    **1. Create src/aurora/cli/ask.py:**

    ```python
    from __future__ import annotations
    import json
    import logging
    import sys
    from typing import Callable

    import typer

    from aurora.llm.prompts import INSUFFICIENT_EVIDENCE_MSG
    from aurora.llm.service import LLMService
    from aurora.retrieval.contracts import RetrievalResult
    from aurora.retrieval.service import RetrievalService

    logger = logging.getLogger(__name__)

    ask_app = typer.Typer(
        no_args_is_help=True,
        help="Pergunte ao vault e receba respostas fundamentadas.",
    )
    ```

    Add a single command registered directly (not as sub-command, since `aurora ask` is top-level):
    Actually, since `aurora ask "query"` must work as a direct command (not `aurora ask ask "query"`), implement it as a standalone command callback:

    ```python
    @ask_app.callback(invoke_without_command=True)
    def ask_command(
        query: str = typer.Argument(..., help="Pergunta para buscar no vault."),
        json_output: bool = typer.Option(False, "--json", help="Renderiza resposta em JSON."),
    ) -> None:
    ```

    Implementation flow:
    1. Create `RetrievalService()` and call `result = retrieval.retrieve(query)` (per D-05: query passed directly, per D-15: always retrieves)
    2. If `result.insufficient_evidence`:
       - Log note paths/scores at DEBUG (per D-07): `logger.debug("Retrieval: %d hits, paths=%s", len(result.notes), [(n.path, n.score) for n in result.notes])`
       - If `json_output`: print `json.dumps({"query": query, "answer": "", "sources": [], "insufficient_evidence": True}, ensure_ascii=False, indent=2)` and return
       - Else: `typer.echo(INSUFFICIENT_EVIDENCE_MSG)` and return
    3. If `result.ok` and not insufficient:
       - Log paths/scores at DEBUG (per D-07): `logger.debug("Retrieval: %d notes, paths=%s", len(result.notes), [(n.path, n.score) for n in result.notes])`
       - Create `LLMService()` and call `response = llm.ask_grounded(query, result.context_text, on_token=_make_token_printer(json_output))`
       - `_make_token_printer(json_output)` returns: if json_output, a no-op lambda that collects tokens into a list; if not json_output, a function that does `print(token, end="", flush=True)` (per Pitfall 7 — never use typer.echo for streaming)
       - After streaming: if not json_output, `print()` for final newline
       - If json_output: print JSON with query, answer (full response text), sources (list of unique note paths from result.notes, per D-09), insufficient_evidence=False
