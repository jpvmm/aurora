<objective>
Increase retrieval recall by raising top-K to 15, doubling context budget to 24K chars, adding a keyword_search() method to QMDSearchBackend, and wiring keyword fallback into RetrievalService for queries containing proper nouns.

Purpose: Current retrieval misses exact-name matches (e.g., "Rosely") because hybrid search weakens on proper nouns. Keyword BM25 search via `qmd search` finds these. Broader top-K and context budget support date-range queries returning many notes.

Output: Updated settings, keyword_search backend method, keyword fallback in all three retrieve methods, passing tests.
</objective>
