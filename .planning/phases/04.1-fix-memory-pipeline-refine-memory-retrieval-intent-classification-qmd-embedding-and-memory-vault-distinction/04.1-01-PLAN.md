<objective>
Upgrade the intent classification from two-way (vault/chat) to three-way (vault/memory/chat), add a memory-first system prompt for memory-intent questions, and restructure the summary format to include structured sections and embedded date for temporal QMD retrieval.

Purpose: The current intent classifier merges memory and vault queries into a single "vault" intent, preventing memory-first retrieval. The current summary format is free-form prose without dates in body text, making temporal queries ("o que conversamos hoje?") unmatchable by QMD semantic search.

Output: Updated prompts, three-way classify_intent, structured summary format with date injection, and all tests updated.
</objective>
