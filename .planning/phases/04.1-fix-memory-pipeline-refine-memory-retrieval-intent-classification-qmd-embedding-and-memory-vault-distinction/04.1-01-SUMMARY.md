## Decisions Made

- Checked "memory" before "vault" in classify_intent() — memory is more specific and checking it first prevents accidental vault matches on memory queries
- Created SYSTEM_PROMPT_MEMORY_FIRST as a separate constant from SYSTEM_PROMPT_GROUNDED_WITH_MEMORY; the former prioritizes session history, the latter prioritizes vault notes
- Date injection placed in MemorySummarizer (not in EpisodicMemoryStore) to keep store.write() signature stable and make date presence an invariant at the domain layer
- SUMMARIZE_SESSION_PROMPT uses {date} and {conversation} placeholders; date.today() called in LLMService.summarize_session() at call time
