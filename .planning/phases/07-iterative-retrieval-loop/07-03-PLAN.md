    **D-08 — Reformulation persistence + LLM-context filtering (BLOCKER FIX):**

    D-08 is locked: "Reformulated query is logged to the chat history as an internal note, NOT shown in the assistant message. `--trace` is the way to see it." Implementation:

    1. **Persist reformulations to chat history (ChatSession side):**
       - In `_handle_vault_turn` and `_handle_memory_turn`, after `iterative.run(...)` returns `(result, trace)` and BEFORE `self._on_trace(trace)`:
         - For each `attempt` in `trace.attempts` where `attempt.attempt >= 2` (i.e. it represents a reformulated query — attempt 1 is the user's original query, never persisted as a reformulation): call `self._history.append_turn("system", f"[reformulation] {attempt.query}")`.
         - Persist BEFORE the user/assistant turns are appended at the end of `process_turn` so the order in the JSONL is: prior turns -> [reformulation 1] -> [reformulation 2] -> user -> assistant. This keeps the historical "thinking" attached to the turn it belonged to.
         - The prefix is the literal ASCII string `[reformulation] ` (open-bracket, lowercase `reformulation`, close-bracket, single space). Used as the filter sentinel by ChatHistory.

    2. **Filter from LLM context (ChatHistory side):**
       - Modify `ChatHistory.get_recent` so it EXCLUDES any record where `role == "system" AND content.startswith("[reformulation] ")`.
       - Filter must apply BEFORE the `max_messages` slicing — so reformulations don't take up slots in the LLM context window (otherwise a chatty turn with 2 reformulations could push real assistant messages out of the recent window).

    3. **Why this design choice:**
       - The on_trace callback is a USER-facing observability hook (it's how `--trace` writes to stderr). Reformulation persistence is a CORE history concern, not observability — putting it inside the on_trace callback would mean disabling --trace also disables persistence. Wrong layering. So persistence happens unconditionally inside `_handle_*_turn`, and the on_trace callback fires SEPARATELY for users who want to see it on stderr.
       - Persistence as `role="system"` (rather than a new role) keeps the JSONL schema unchanged and avoids breaking existing readers. The `[reformulation] ` prefix is the contract.
       - Filtering in `get_recent` (rather than in chat-prompt assembly) means EVERY consumer of get_recent benefits — the filter is at the source of LLM-bound history, not at every call site.

       ```python
       """Render helpers for RetrievalTrace (text + dict).

       Pure functions — no I/O, no formatting state. Privacy-safe by construction
       because RetrievalTrace and RetrievalAttempt structurally exclude content
       fields (PRIV-03, see test_trace_contracts.py).
       """
       from __future__ import annotations

       import dataclasses
       from typing import Any

       from aurora.retrieval.contracts import RetrievalAttempt, RetrievalTrace


       _PATH_PREVIEW_LIMIT = 3


       def _format_paths(paths: tuple[str, ...]) -> str:
           if not paths:
               return "(nenhum)"
           if len(paths) <= _PATH_PREVIEW_LIMIT:
               return ", ".join(paths)
           preview = ", ".join(paths[:_PATH_PREVIEW_LIMIT])
           return f"{preview}, ... (+{len(paths) - _PATH_PREVIEW_LIMIT})"


       def _format_attempt(att: RetrievalAttempt, *, is_last: bool) -> str:
           verdict_word = "ok" if att.verdict_ok else "thin"
           lines = [
               f"  [{att.attempt}] q={att.query!r} strategy={att.strategy} "
               f"hits={att.hit_count} top={att.top_score:.2f} verdict={verdict_word} "
               f"({att.duration_ms:.0f}ms)",
           ]
           if not att.verdict_ok and att.verdict_signals:
               lines.append(f"      signals: {'; '.join(att.verdict_signals)}")
           lines.append(f"      paths: {_format_paths(att.paths)}")
           if att.reformulation_skipped:
               lines.append(
                   f"      reformulation: skipped ({att.reformulation_reason})"
                   if att.reformulation_reason
                   else "      reformulation: skipped"
               )
           elif not is_last and not att.verdict_ok and att.reformulation_reason:
               lines.append(f"      reformulation: {att.reformulation_reason}")
           return "\n".join(lines)


       def render_trace_text(trace: RetrievalTrace) -> str:
           """Render a human-readable multi-line trace summary."""
           if not trace.enabled:
               return "[trace] disabled (iterative loop off)"
           final_word = "ok" if trace.final_verdict_ok else "thin"
           judge_word = "enabled" if trace.judge_used else "disabled"
           header = (
               f"[trace] intent={trace.intent} attempts={len(trace.attempts)} "
               f"final={final_word} duration={trace.total_duration_ms:.0f}ms "
               f"judge={judge_word}"
           )
           if not trace.attempts:
               return header
           body_lines = [
               _format_attempt(att, is_last=(i == len(trace.attempts) - 1))
               for i, att in enumerate(trace.attempts)
           ]
           return "\n".join([header, *body_lines])


       def render_trace_dict(trace: RetrievalTrace) -> dict[str, Any]:
           """Render trace as a plain dict for JSON envelope embedding."""
           return dataclasses.asdict(trace)


       __all__ = ["render_trace_text", "render_trace_dict"]
       ```

       ```python
       #!/usr/bin/env python3
       """Measure IterativeRetrieval latency vs single-shot baseline (D-12, Pitfall 6).

       Usage:
           python scripts/measure_iterative_retrieval_latency.py [--iterations N] [--query "..."]

       Requires a live llama.cpp endpoint + a populated qmd vault. For each query,
       runs the loop twice — once with iterative_retrieval_enabled=True, once with
       False — and reports the enabled/disabled RATIO per query plus a final
       WITHIN/OVER BUDGET verdict against the D-12 targets (happy <= 1.0x,
       worst <= 2.5x).
       """
       from __future__ import annotations

       import argparse
       import statistics
       import sys
       import time
       from dataclasses import dataclass
       from typing import Callable

       from aurora.llm.service import LLMService
       from aurora.memory.store import MEMORY_COLLECTION, MEMORY_INDEX
       from aurora.retrieval.iterative import IterativeRetrieval
       from aurora.retrieval.qmd_search import QMDSearchBackend
       from aurora.retrieval.service import RetrievalService
       from aurora.runtime.settings import RuntimeSettings, load_settings


       @dataclass
       class _PairMeasurement:
           query_label: str
           iteration: int
           disabled_ms: float
           enabled_ms: float
           ratio: float
           enabled_attempts: int
           enabled_final_ok: bool


       _DEFAULT_QUERIES = [
           # (label, query). The "happy" one should hit on attempt 1 if your vault
           # has any project-related note. Adjust if your vault is empty.
           ("happy", "Aurora"),
           ("vague", "notas sobre saude"),
           ("no-match", "xpto qwerty zzz nonsense"),
       ]

       _D12_HAPPY_TARGET = 1.0   # <= 1x single-shot on happy path
       _D12_WORST_TARGET = 2.5   # <= 2.5x single-shot on worst case


       def _make_settings_loader(*, enabled: bool, base: RuntimeSettings) -> Callable[[], RuntimeSettings]:
           """Return a loader that yields settings identical to the user's defaults
           except for the iterative_retrieval_enabled flag.
           """
           overrides = base.model_dump()
           overrides["iterative_retrieval_enabled"] = enabled
           return lambda: RuntimeSettings(**overrides)


       def _time_one(orch: IterativeRetrieval, retrieval: RetrievalService, query: str) -> tuple[float, int, bool]:
           """Run orchestrator once, return (total_ms, attempts, final_ok)."""
           t0 = time.monotonic()
           first = retrieval.retrieve_with_memory(query)
           def _retrieve_fn(q, s, t):
               return retrieval.retrieve_with_memory(q, search_strategy=s, search_terms=t)
           _result, trace = orch.run(
               query, retrieve_fn=_retrieve_fn, strategy="hybrid", terms=[],
               intent="vault", first_attempt=first,
           )
           return (time.monotonic() - t0) * 1000.0, len(trace.attempts), trace.final_verdict_ok


       def main() -> int:
           p = argparse.ArgumentParser(description=__doc__)
           p.add_argument("--iterations", type=int, default=5,
                          help="Iterations per query (default 5)")
           p.add_argument("--query", action="append",
                          help="Override default queries; can be passed multiple times. Label becomes 'custom'.")
           args = p.parse_args()

           queries = (
               [("custom", q) for q in args.query]
               if args.query
               else _DEFAULT_QUERIES
           )

           try:
               base_settings = load_settings()
               memory_backend = QMDSearchBackend(
                   index_name=MEMORY_INDEX, collection_name=MEMORY_COLLECTION,
               )
               retrieval = RetrievalService(memory_backend=memory_backend)
               llm = LLMService()
               enabled_orch = IterativeRetrieval(
                   retrieval=retrieval, llm=llm,
                   settings_loader=_make_settings_loader(enabled=True, base=base_settings),
               )
               disabled_orch = IterativeRetrieval(
                   retrieval=retrieval, llm=llm,
                   settings_loader=_make_settings_loader(enabled=False, base=base_settings),
               )
           except Exception as exc:  # noqa: BLE001
               print(
                   f"ERRO: nao foi possivel inicializar retrieval/LLM. "
                   f"Verifique `aurora model status`. Detalhe: {exc}",
                   file=sys.stderr,
               )
               return 2

           print("query_label,iteration,disabled_ms,enabled_ms,ratio,attempts,final_ok")
           rows: list[_PairMeasurement] = []
           for label, q in queries:
               for i in range(1, args.iterations + 1):
                   try:
                       d_ms, _, _ = _time_one(disabled_orch, retrieval, q)
                       e_ms, e_att, e_ok = _time_one(enabled_orch, retrieval, q)
                   except Exception as exc:  # noqa: BLE001
                       print(f"ERRO durante medicao ({label} #{i}): {exc}", file=sys.stderr)
                       continue
                   ratio = e_ms / d_ms if d_ms > 0 else float("inf")
                   row = _PairMeasurement(
                       query_label=label, iteration=i, disabled_ms=d_ms,
                       enabled_ms=e_ms, ratio=ratio, enabled_attempts=e_att,
                       enabled_final_ok=e_ok,
                   )
                   rows.append(row)
                   print(
                       f"{row.query_label},{row.iteration},{row.disabled_ms:.1f},"
                       f"{row.enabled_ms:.1f},{row.ratio:.2f},"
                       f"{row.enabled_attempts},{row.enabled_final_ok}"
                   )

           # Per-query summary
           print("", flush=True)
           print("# Per-query summary (median ms, median ratio, p95 ratio)", flush=True)
           by_label: dict[str, list[_PairMeasurement]] = {}
           for r in rows:
               by_label.setdefault(r.query_label, []).append(r)

           happy_median_ratio: float | None = None
           worst_p95_ratio: float = 0.0
           for label, group in by_label.items():
               if not group:
                   print(f"# {label}: (no measurements)")
                   continue
               disabled_med = statistics.median(g.disabled_ms for g in group)
               enabled_med = statistics.median(g.enabled_ms for g in group)
               ratios = sorted(g.ratio for g in group)
               ratio_med = statistics.median(ratios)
               p95_idx = max(0, int(round(0.95 * (len(ratios) - 1))))
               ratio_p95 = ratios[p95_idx]
               print(
                   f"# {label}: n={len(group)} disabled_med={disabled_med:.0f}ms "
                   f"enabled_med={enabled_med:.0f}ms ratio_med={ratio_med:.2f}x "
                   f"ratio_p95={ratio_p95:.2f}x"
               )
               if label == "happy":
                   happy_median_ratio = ratio_med
               worst_p95_ratio = max(worst_p95_ratio, ratio_p95)

           # D-12 verdict
           print("", flush=True)
           print("# D-12 verdict", flush=True)
           if happy_median_ratio is None:
               print("# Happy-path ratio: (no `happy` query measurements; cannot verdict)")
           else:
               verdict_h = (
                   "WITHIN BUDGET" if happy_median_ratio <= _D12_HAPPY_TARGET
                   else "OVER BUDGET"
               )
               print(
                   f"# Happy-path ratio: {happy_median_ratio:.2f}x "
                   f"(target <= {_D12_HAPPY_TARGET:.1f}x) [{verdict_h}]"
               )
           verdict_w = (
               "WITHIN BUDGET" if worst_p95_ratio <= _D12_WORST_TARGET
               else "OVER BUDGET"
           )
           print(
               f"# Worst-case ratio (max p95 across queries): {worst_p95_ratio:.2f}x "
               f"(target <= {_D12_WORST_TARGET:.1f}x) [{verdict_w}]"
           )
           return 0


       if __name__ == "__main__":
           sys.exit(main())
       ```
